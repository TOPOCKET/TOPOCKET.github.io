# 归档索引

本页用于快速定位历史文档，按“主归档 + 专项归档”组织。

## 主归档

- [completed-work-archive.md](./completed-work-archive.md)  
  已完成优化路线与实施记录（诸神优化主线、A* 固化、清理动作、验收结果）。

## 专项归档

  可观测性专项记录（O 路线与诊断链路建设）。

## 使用建议

1. 查“最近完成了什么”优先看 `completed-work-archive.md`。  
2. 查“某条专项路线细节”再进入对应专项归档。  
3. README 中的临时优化路线在阶段完成后迁移到主归档。
